# Dharaneeshwar's Portfolio

A personal portfolio website showcasing my professional experience, academic achievements, technical skills, and projects as a Software Engineer and Graduate Student at the University at Buffalo.

## Live Demo
[https://dharanee.dev//](https://dharanee.dev/))

## 🛠️ Built With
This project is built using modern web development technologies, heavily optimized for speed and a lightweight bundle:
- **React 18** (with TypeScript)
- **Vite** (Next Generation Frontend Tooling)
- **Tailwind CSS** (Utility-first styling framework)
- **React Router** (Client-side routing)
- **Radix UI Primitive Components**
- **Lucide React** (Icons)

## ⚡ Features
- **Fully Responsive:** Beautifully crafted for Mobile, Tablet, and Desktop.
- **Dark/Light Mode:** Seamless theme toggling powered by custom contexts.
- **Optimized Bundle:** Lazily-loaded routes and completely stripped of unused dependency bloat.
- **Scroll Animations:** Intersection Observer APIs built-in for smooth fade-ins.

## 💻 Running Locally

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/dharaneesk/myeportfolio.git
   ```
2. Navigate into the directory:
   ```bash
   cd myeportfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development Server
Run the local Vite development server:
```bash
npm run dev
```

### Production Build
To create an optimized production build:
```bash
npm run build
```
This will output the static files into the `dist/` directory, which can be deployed to GitHub Pages or any static hosting service.
